Добавление видео

Видео добавляется со следующих сайтов:
www.myvi.tv - при добавлении недавно добавленных на данный ресурс видео может возникнуть проблема. Если вы скопировали URL видео из плеера, но при добавлении выскакивает красное окно об ошибке, то необходимо при нажатии ПКМ выбрать пункт "Открыть видео в новой вкладке" и скопировать ссылку из адресной строки браузера в новой вкладке.

sibnet.ru - Под плеером будет иметься кнопка шестерёнки . Её необходимо нажать и выбрать способ буферизации "Полностью". После этого жмём ПКМ по видео и выбираем "Открыть видео в новой вкладке". И копируем ссылку по аналогии с myvi.tv, т.е. из адресной строки браузера в новой вкладке (в правильной ссылке на конце должно содержаться "noip=1"). Внимание: ссылки с сибнета работают в течении нескольких часов, после чего они перестают открываться и необходимо получать новую ссылку.

animevost.club/ - На сайте имеется кнопка скачивания. Просто кликаем по ней ПКМ и нажимаем "скопировать URL (ссылку)". Извлечённые ссылки не всегда годятся для добавления, но попробовать стоит если нет других вариантов.

Базы видео с аниме:
findanime.org - Содержит в себе плееры с сайтов sibnet и myvi. Также есть плеер с сайта kiwi, из которого тоже можно извлечь ссылку и добавить (но скорость низкая и видео будет зависать).
animespirit.ru - Содержит в себе плееры с сайтов sibnet и myvi (в основном старый сайт myvi который заблокирован РосКомНадзором. Поэтому можно добавлять только новые тайтлы 2018 года начиная с осеннего сезона). -Перезалив старых тайтлов!

Общие правила по добавлению.
Чтобы добавить видео в плейлист нужно нажать кнопку "+" и вставить ссылку. Для добавления названия необходимо вызвать ещё одну строку. Чтобы это сделать необходимо в момент когда курсор активен в строке с добавлением видео нажать кнопку "вниз" на клавиатуре. Тогда снизу появится ещё одна строка в которую нужно будет ввести название.
Способ 1. На большинстве сайтов достаточно просто кликнуть по видео правой кнопкой мыши и нажать "Скопировать URL".
Способ 2. Для удобства можно скачать расширение для браузера, которое будет обнаруживать ссылки автоматически и их можно будет легко скопировать. Например, для браузеров Chrome, Firefox и Opera можно скачать расширение Video Downloader Prime. Данный аддон будет полезен для быстрого извлечения ссылок с сайта seasonvar.ru и некоторых других.
